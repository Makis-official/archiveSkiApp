графия

Стеганография — это метод скрытой передачи информации путем встраивания данных в другой носитель без видимых изменений.
Принцип работы стеганографии при встраивании информации в текст заключается в скрытии данных внутри текстового файла так, чтобы изменения оставались незаметными для человека. Это достигается за счет использования различных техник, которые не влияют на читаемость текста, но позволяют передать скрытую информацию.

Задание: Найдите скрытое сообщение внутри текста

Ключ: 3

Текст: Хвйзос е цъзрлл - озёнс е дсб

Введите расшифрованное высказывание
