атбаш

Атбаш — это древний моноалфавитный шифр подстановки, где первая буква алфавита заменяется на последнюю, вторая — на предпоследнюю и так далее.
Этот метод восходит к библейским временам — его использовали ещё древние евреи для зашифровывания слова "Вавилон" в Книге Иеремии, а название "Атбаш" образовано от сочетания первых (Алеф-Тав) и последних (Бет-Шин) букв еврейского алфавита.
