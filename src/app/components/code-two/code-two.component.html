азбука морзе

Азбука Морзе — способ знакового кодирования, в котором буквы алфавита, цифры, знаки препинания и другие символы представляются в виде последовательностей коротких и длинных сигналов, называемых «точками» и «тире». Предназначена для передачи по последовательным каналам связи.
Во время Второй мировой войны азбука Морзе стала одним из основных способов передачи информации. Она позволяла принимать сигнал на дальних расстояниях и в условиях сильных радиопомех, кодировать послания вручную, а запись и воспроизведение происходили с помощью простейших устройств.
